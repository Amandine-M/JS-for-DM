SETI partie 2

Brief

Des documents viennent d’être dérobé, un espion est parmi nous!

Par chance, une caméra de surveillance a pu filmer la silhouette de l’espion avant qu’il ne disparaisse :

  • C’est un homme,
  • de grande taille (1.80 ou plus)
  • et qui a les yeux verts.

Vous devez développer une fonction qui permette de rechercher les personnes correspondantes à la description ci dessus parmi la liste transmise.

Au final, vous devez avec une short list de 4 suspects

À la suite de cet énoncé, vous trouverez, sur la plateforme, les sources suivantes dans un fichier téléchargeable.

DATA.JS
Le fichier “data.js” contient la liste des 100 personnes qui ont eu accès au bâtiment, avec leurs contacts qui ont pu être invités sur les lieux.
L’espion se trouve parmi elle. Les données sont structurées en JSON
CHASSE_AUX_ESPIONS.HTML
Le fichier ou vous devez écrire le code nécessaire.

On affichera le résultat dans la page.

Résultat

Liste des suspects Afficher/masquer

    Conseils

    1. Lisez bien toutes les informations et fichiers transmis
    2. Utilisez le serveur local “live-server” pour tester l’exercice
    3. Notez bien la structure du JSON pour pouvoir le parcourir dans sa totalité.
